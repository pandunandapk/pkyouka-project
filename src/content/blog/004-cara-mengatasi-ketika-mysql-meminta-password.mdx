Ketika kita mengalami error seperti ini mungkin sering ditemui ketika kita sedang melakukan
koneksi ke database mysql. Error ini biasanya terjadi ketika kita sedang melakukan koneksi ke databasenya
tanpa menggunakan password.

```bash
sudo mysql -u root
```

ketika sudah masuk ke mysqlnya maka ketikkan perintah berikut ini

```bash
USE mysql;
UPDATE user SET plugin='mysql_native_password' WHERE User='root';
FLUSH PRIVILEGES;
exit;
```

Setelah itu restart mysqlnya dengan perintah berikut ini

```bash
sudo service mysql restart
```

maka ketika kita coba untuk login kembali ke mysqlnya maka kita tidak akan diminta password lagi.
Silahkan dicoba dan semoga berhasil :smile:
